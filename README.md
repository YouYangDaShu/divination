# 🔮 玄机阁 - 中国传统占卜系统

一个基于 Flask 的中国传统占卜网页应用，支持六爻、小六壬、八字、紫微斗数、梅花易数、奇门遁甲、黄历查询、解梦等多种占卜方式。

## ✨ 功能特点

### 占卜方式
- 🔮 **六爻卦** - 铜钱起卦，完整卦象解析
- 🎯 **小六壬** - 快速占卜，适合日常小事
- 📅 **八字排盘** - 年月日时四柱分析
- ⭐ **紫微斗数** - 星曜排列，命盘解析
- 🌸 **梅花易数** - 数字起卦，灵活便捷
- 🗺️ **奇门遁甲** - 时家奇门，方位分析
- 📖 **黄历查询** - 每日宜忌，吉时吉方
- 💭 **解梦** - 周公传统 + 心理学双视角

### 特色功能
- 🤖 **AI 智能解读** - 结合用户处境的白话解读（可选）
- 📊 **统计面板** - 查看占卜使用统计
- 💾 **Obsidian 集成** - 自动保存占卜记录到笔记
- 🎨 **精美界面** - 深色主题，响应式设计
- ⚡ **本地排盘** - 核心计算完全本地，无需联网
- 🔒 **隐私安全** - 数据存储在本地，不上传服务器

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install flask requests
```

### 2. 配置 AI（可选）

如果你想使用 AI 解读功能，需要配置 API：

```bash
# 复制配置文件
cp config.example.py config.py

# 编辑 config.py，填入你的 API 配置
```

支持的 AI 服务：
- OpenAI (GPT-3.5/4)
- DeepSeek
- 本地模型 (Ollama 等)
- 其他兼容 OpenAI API 的服务

### 3. 启动应用

```bash
python app.py
```

访问 http://127.0.0.1:5066

## 🛠️ 技术栈

- **后端**: Python Flask
- **前端**: HTML + CSS + JavaScript
- **存储**: SQLite (统计) + 文件 (Obsidian)
- **AI**: OpenAI 兼容 API (可选)

## 📝 使用说明

### 基础占卜
1. 选择占卜方式（六爻、小六壬等）
2. 输入问题（可选）
3. 点击"开始占卜"
4. 查看排盘结果

### AI 解读
1. 填写"心中所问"或"我的处境"
2. 系统会自动调用 AI 进行解读
3. AI 解读是可选功能，不影响排盘

### 黄历查询
1. 点击"黄历查询"
2. 选择日期
3. 查看当日宜忌、吉时、冲煞等
